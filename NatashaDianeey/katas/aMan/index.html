KATAS

Cada mañana, un hombre camina al trabajo, y cada tarde vuelve a casa.

Si está lloviendo por la mañana y tiene un paraguas en casa, toma un paraguas para el viaje para que no se moje y lo almacena en el trabajo. Del mismo modo, si está lloviendo por la tarde y tiene un paraguas en el trabajo, toma un paraguas para el viaje a casa.

Dada una serie de condiciones climáticas, su tarea es calcular el número mínimo de paraguas que necesita para comenzar a fin de que nunca se moje. Puede comenzar con algunos paraguas en casa y otros en el trabajo, pero la salida es un solo número entero, el número total mínimo.

La entrada es una matriz / lista de pronósticos meteorológicos de medio día consecutivos. Así que, por ejemplo el primer valor es el clima de la mañana del primer día y el segundo valor es el clima de la tarde del primer día. Las opciones son "despejado", "soleado", "nublado", "lluvioso", "ventoso" o "tormentas eléctricas".

p.ej. Durante tres días, 6 valores:

weather = ["rainy", "cloudy", "sunny", "sunny", "cloudy", "thunderstorms"]

N.B. He never takes an umbrella if it is not raining.

Examples:

minUmbrellas(["rainy", "clear", "rainy", "cloudy"])
// should return 2
// Because on the first morning, he needs an umbrella to take, and he leaves it at work.
// So on the second morning, he needs a second umbrella.

minUmbrellas(["sunny", "windy", "sunny", "clear"])
// should return 0
// Because it doesn't rain at all

minUmbrellas(["rainy", "rainy", "rainy", "rainy", "thunderstorms", "rainy"])
// should return 1
// Because he only needs 1 umbrella which he takes on every journey.
